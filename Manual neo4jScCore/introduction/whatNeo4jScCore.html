Что такое neo4jScCore?

   Neo4jScCore – это библиотека, написанная на языке программирования Java с использование графовой базы данных Neo4j(www.neo4j.org).
   Библиотека предоставляет простой интерфейс для создания, хранения и поиска данных, представленных в виде семантических сетей.
   При помощи данной библиотеки можно создать практически любую достаточно сложную семантическую сеть, потратив на это минимум усилий)))
