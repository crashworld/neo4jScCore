Узлы


В библиотеке neo4jScCore  все узлы являются объектами класса ScNode. Для узла мы можем:

  • просмотреть и изменить его имя (getName(); setName(nodeName))
  • просмотреть, изменить и удалить  тип узла(addType(type); addTypes(types); removeType(type))
  • просмотреть и установить контент узла(getContent(); setContent(content))
  • узнать входящие, исходящие дуги для данного узла(getAllOutputScArcs(); getAllInputScArcs() и другие…).

Более детальную информацию о методах можно найти в javadoc.

Пример (Просмотр и Изменение  имени узла):

String nodeName = “first”;
scNode.setName(nodeName);


Пример (Изменение и удаление типа узла):

List<String> nodeConstType = new ArrayList<String>();
        nodeConstType.add(ScElementTypes.CONST);
        nodeConstType.add(ScElementTypes.NODE);

//addTypes
scNode.addTypes(nodeConstType);

//remove CONST Type
scNode.removeType(ScElementTypes.CONST);
