Узлы


В библиотеке neo4jScCore  все узлы являются объектами абстрактного класса ScNode. Реализация же метов класса ScNode находится непосредственно в классе ScNodeImpl. SC узлы представляются в памяти как обычные узлы базы данных neo4j. Имя узла задаётся как значение свойства _scNodeName. Для узла мы можем:

  • установить, просмотреть и изменить его имя
  • добавить, просмотреть, удалить  тип узла
  • просмотреть и установить содержимое узла
  • узнать входящие, исходящие дуги для данного узла.

Более детальную информацию о методах можно найти в javadoc.

К примеру установим имя для узла:

scNode.setName("first");


Затем установим и удалим тип узла:

List<String> nodeConstType = new ArrayList<String>();
nodeConstType.add(ScElementTypes.CONST);
nodeConstType.add(ScElementTypes.NODE);

scNode.addTypes(nodeConstType);

//remove CONST Type
scNode.removeType(ScElementTypes.CONST);
